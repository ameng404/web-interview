# 项目介绍

Web Interview 是一个由语音打卡社群（JS 每日一题）维护的前端面试题宝典，基于 [Rspress](https://rspress.rs/) 构建，覆盖 15 大前端技术专题，收录 200+ 精选面试题及详细题解。

## 内容覆盖

本站面试题按技术方向划分为以下专题：

| 分类 | 专题 | 题数 |
|------|------|------|
| 框架 | Vue、Vue3、React | 71 |
| 语言 | JavaScript、ES6、TypeScript | 57 |
| 样式 | CSS | 20 |
| 工程化 | Webpack | 10 |
| 网络 | HTTP | 14 |
| 服务端 | Node.js | 14 |
| 工具 | Git、操作系统 | 17 |
| 其他 | 算法、小程序、设计模式 | 32 |

## 适合人群

- 准备前端面试的求职者
- 想要系统复习前端知识的开发者
- 带团队面试的面试官，需要面试题参考
- 前端学习者，通过面试题反推知识点

## 如何使用

左侧导航栏点击「面试题」即可进入题库，侧边栏按技术专题分类，每个专题可展开/折叠，点击具体题目查看详细题解。

你也可以使用页面右上角的搜索功能快速定位你关心的问题。
