# 代码块

面试题解中经常需要展示代码示例。本站使用 [Shiki](https://shiki.style) 进行编译时语法高亮，支持大量编程语言。

## 基本用法

使用三个反引号加语言标识来创建代码块：

```js
const arr = [1, 2, 3];
const doubled = arr.map(item => item * 2);
console.log(doubled); // [2, 4, 6]
```

## 支持的语言

常用语言标识：`js`、`ts`、`jsx`、`tsx`、`css`、`html`、`bash`、`json`、`md` 等。

完整列表参考 [Shiki 支持的语言](https://shiki.style/languages)。
